A) HTML Structure - top

Here have 5 different templates. All templates are fixed 2 columns layout. All pages html structure are same unless few small changes. Here is the general html structure.

If you would like to customize any templates then you can easily do it. Here I am describing about all html portion with images of these templates. Here two div contain all content of the template. "wrapper" Div and "container" Div. In body tag I used Id=''error404" I always suggest to use this id withi body tag of this template html because of it will not conflict your website others html/css. "container" Div have two column leftSide and rightSide. Here have also "branding" div contains logo or brand name of your site where you will use this template.

lieftSide contain following elements:

  • Error code - (class="errorCode" here I use 206 px size text you can edit here and you can put any error code here like 403, 306)
  • A bubble with error message. - ( class="bubble" You can put any relevant error message here.)

B) CSS Files and Structure - top

I used one css styles.css file you can change this name if you need or you can copy all code and paste it to your site css file. If you would like to edit a specific section of the page, simply find the appropriate class/id in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Here I used windows default font name Arial and I used a google web font name 'Oswald' for error code and heading/error message. If you don't want to use any google web font then pllease remove <link href='http://fonts.googleapis.com/css?family=Oswald' rel='stylesheet' type='text/css'> this line from html head and change appropriate font family in css.

Here is css structure:
